# 基于uni-app的仿网易云音乐网站

<hr/>

> 一个仿网易云音乐的移动端音乐网站

## 1. 功能

* 首页
  - 轮播图
  - 编辑推荐（歌单）
  - 最新音乐
* 搜索
  - 热门搜索
  - 搜索建议
  - 搜索详情
* 歌曲详情
  - 歌曲播放
  - 歌曲评论
* 热歌榜
* 推荐MV
* MV详情
  - MV播放
  - MV评论
* 歌手详情
